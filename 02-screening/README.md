# 02 — Screening & Scorecard

Méthodologie d'évaluation structurée développée et utilisée chez Fnac Darty.  
Objectif : décision sur données, réduction des biais cognitifs.

---

## Architecture — 4 blocs, 17 critères

| Bloc | Contenu | Poids |
|---|---|---|
| A | Compétences techniques (Hard Skills) | 35 % |
| B | Soft Skills | 25 % |
| C | Culture & Valeurs | 20 % |
| D | Motivation & Projet | 20 % |

Échelle de notation : **1 à 4** (validation de données active dans Excel, seules ces valeurs sont acceptées)

---

## Grille de décision automatique

| Score global /4 | Décision |
|---|---|
| ≥ 3,5 | ✅ Validation forte — avancer |
| 3,0 – 3,4 | 🟡 Validation conditionnelle — panel requis |
| 2,5 – 2,9 | 🔶 Réserves — arbitrage collégial |
| < 2,5 | ❌ Non-sélection — documenter le motif |

---

## Critères éliminatoires

> Si un critère éliminatoire reçoit un score < 2, le candidat est éliminé quel que soit son score global.

Exemples sur poste ADV B2B (Fnac Darty) :

- Critère 1 — Maîtrise ADV / O2C
- Critère 4 — Recouvrement
- Critère 6 — Rigueur
- Critère 12 — Intégrité
- Critère 17 — Motivation / adhésion au projet

---

## Mode opératoire pas à pas

**Étape 1** — Renseigner : nom candidat, date, évaluateur, étape (RH / Manager / N+2)

**Étape 2** — Saisir les scores colonne E (1 à 4). Commentaire obligatoire si score ≤ 2.

**Étape 3** — Lire les résultats automatiques :
- Score moyen par bloc
- Score pondéré par bloc (moyenne × pondération)
- Score global = somme des scores pondérés
- Décision et alerte éliminatoire générées automatiquement

**Étape 4** — Rédiger la recommandation finale : décision, points forts, vigilances, prochaine étape.

---

## Gouvernance multi-évaluateurs

| Rôle | Blocs évalués |
|---|---|
| RH (qualification téléphonique) | B + D |
| Manager N+1 | A + B + C |
| N+2 / Direction | Score global + D |

> Règle absolue : chaque évaluateur remplit **sa propre copie** du fichier.  
> En débriefing panel : partager l'onglet Comparatif, pas les scorecards individuelles.

---

## Règles RGPD

- Ne pas partager le fichier hors RH + évaluateurs concernés
- Ne pas conserver plus de 2 ans après fin de process
- Tout critère doit être exclusivement professionnel (Art. L.1132-1 Code du travail)
- Le candidat peut demander accès à son évaluation (droit RGPD)

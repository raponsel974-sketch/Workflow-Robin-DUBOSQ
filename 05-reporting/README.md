# 05 — Reporting & KPIs

Structure de pilotage recrutement utilisée sur l'ATS Excel V12 (Fnac Darty).

---

## KPIs suivis

| KPI | Définition | Cible |
|---|---|---|
| TTH (Time to Hire) | Brief → promesse signée | ≤ 30 jours |
| Taux conversion sourcing | Profils identifiés → entretien RH | > 15 % |
| Taux conversion RH→Manager | Envoyés manager / qualifiés RH | > 60 % |
| Taux sélection finale | Embauches / Entretiens manager | > 30 % |
| Taux refus candidat | Offres refusées / Offres faites | < 10 % |
| NPS candidat | Satisfaction expérience process | > 7/10 |

---

## Entonnoir de conversion cible
```
Identifié
  ↓ > 50%
Envoyé au manager
  ↓ > 70%
Entretien Manager
  ↓ > 50%
Embauché ✅
```

Les abandons et refus sont tracés à chaque étape pour identifier où se produisent les pertes.

---

## Graphiques automatiques du dashboard ATS

1. **Application Source** — volume de candidatures par canal
2. **Final Decision** — répartition des finalités (embauché / refusé / retiré / stand by)
3. **Recruitment Pipeline** — entonnoir de conversion
4. **Monthly Applications** — tendance temporelle

Filtres disponibles : recruteur · département · source · statut poste · période
